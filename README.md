# OCR Agent with Tesseract

This project uses **Tesseract OCR** for optical character recognition (OCR) from images.  
Before running the agent, make sure the required system dependencies are installed.

---

> [!IMPORTANT]
> The agent requires some external dependencies to work properly.  
> Please install the following packages on your system:
>
> ```bash
> sudo apt install tesseract-ocr
> sudo apt install libtesseract-dev
> ```
>
> If you want to enable **Spanish** text recognition, install the additional language package:
>
> ```bash
> sudo apt install tesseract-ocr-spa
> ```
